## 1. Быстрый старт

IconForge поставляется в виде одного автономного исполняемого файла (`IconForge_v1.0.2_win-x64.exe`). Приложение не требует установки, прав администратора или предварительно установленных библиотек. Для запуска просто дважды кликните по `.exe` файлу.

---

## 2. Выбор и загрузка файлов

Загрузить исходный файл можно любым из следующих способов:
* **Drag-and-Drop:** Перетащите файл прямо в главную область загрузки программы.
* **Обзор:** Нажмите на иконку папки, чтобы выбрать файл через системный проводник Windows.
* **Контекстное меню:** Нажмите правой кнопкой мыши на файл PNG, SVG или ICO в Проводнике Windows и выберите **«Сгенерировать иконки в IconForge»**.

### Поддерживаемые форматы
* **PNG (`.png`):** Растровые изображения высокого разрешения (рекомендуется от 1024x1024).
* **SVG (`.svg`):** Векторная графика (растеризуется с идеальной четкостью для всех размеров).
* **ICO (`.ico`):** Файлы иконок ICO (активирует режим обратного извлечения PNG-кадров).

## 3. Фильтры и коррекция изображения

В левой панели находятся настройки фильтров и оформления:

* **Яркость (-100..+100):** Плавная регулировка яркости изображения без клиппинга.
* **Контраст (-100..+100):** Изменение контрастности картинки.
* **Скругление (%) (0..50%):** Настройка радиуса скругления углов иконки.
* **Отступ (%) (0..40%):** Внутренний отступ логотипа от краев иконки.
* **Черно-белый режим:** Преобразование цветов в монохромный режим.
* **Инверсия цветов:** Инвертирование цветовых каналов RGB.
* **Тень (Drop Shadow):** Добавление мягкой современной тени вокруг силуэта логотипа.
* **Цветовой оттенок (SVG Tint):** Ввод Hex-кода цвета (например, `#0078D4` или `#3DDC84`) для перекраски векторных или однотонных логотипов.

## 4. Форматы экспорта и назначение

Выберите нужные галочки в правой панели экспорта:

* **Windows Classic `.ico`:** Создает мультиформатный файл `app_icon.ico` с разрешениями `16x16`, `24x24`, `32x32`, `48x48`, `64x64`, `128x128` и `256x256` пикселей с микро-шарпингом мелких размеров.
* **Windows Modern Assets:** Генерирует набор отдельный PNG-логотипов для манифестов UWP/WinUI (`Square44x44Logo`, `Square150x150Logo`, `StoreLogo`) во всех масштабах от `scale-100` до `scale-400`.
* **Web & Favicon Pack:** Генерирует комплект для сайтов: `favicon.ico`, `favicon-16x16.png`, `favicon-32x32.png`, `apple-touch-icon.png` (180x180), `android-chrome-192x192.png`, `android-chrome-512x512.png` и манифест `site.webmanifest`.
* **macOS App Icon (`.icns`):** Сборка бинарного файла иконки Apple `.icns`, содержащего все слои для macOS (`ic04` — `ic10`).
* **Android Adaptive & Legacy Icons:** Генерация слоев `Foreground.png` (safe zone 72dp) и `Background.png`, структуры папок `mipmap` (`mdpi` — `xxxhdpi`), круглой иконки `ic_launcher.png` и промо-изображения Google Play `512x512`.

## 5. Сетка живого предпросмотра (Live Preview)

В правой части экрана расположена сетка живого предпросмотра, отображающая иконку в реальных размерах `16x16`, `32x32`, `48x48` и `256x256` пикселей.

### Переключение фона
Используйте кнопки переключения фона в шапке сетки для проверки читаемости иконки:
1. **Прозрачный:** Шахматная сетка прозрачности.
2. **Темный:** Темный фон `#1F1F1F` (имитация темной панели задач Windows 11).
3. **Светлый:** Светлый фон `#F3F3F3`.
4. **Цвет из настроек Android:** Фоновый цвет, выбранный в блоке настройки фона Android.

## 6. Обратное извлечение кадров из ICO

При открытии любого `.ico` файла:
1. Над кнопкой генерации появляется карточка **«Извлечение кадров из ICO»**.
2. IconForge определяет количество и размеры зашитых внутрь PNG/BMP слоев.
3. Нажатие **«Извлечь все кадры в PNG»** сохраняет каждый слой в отдельный файл PNG.

---

## 7. Системная интеграция и Проводник

* **Контекстное меню Проводника:** Переключатель **«Контекстное меню Windows»** регистрирует пункт в `HKEY_CURRENT_USER\Software\Classes\*\shell\IconForge` без запроса прав администратора.
* **Всплывающие уведомления:** По завершении нарезки Windows 11 показывает уведомление с интерактивной кнопкой «Открыть папку».

---

## 8. Одиночный исполняемый файл (Single-File)

В версии v1.0.2 таблица ресурсов `resources.pri` встроена непосредственно внутрь бинарного `.exe`. При запуске `App.xaml.cs` автоматически извлекает нужные ресурсы в рабочую папку. Это позволяет запускать `IconForge_v1.0.2_win-x64.exe` в любой директории под любым именем.
